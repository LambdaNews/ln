## Build Setup

**Requires Node.js 7 and yarn+**

``` bash
# install dependencies
yarn && yarn setup

# serve in dev mode with hot reload at localhost:1234
yarn dev

# build for production
yarn bundle

# deploy to web
yarn deploy
```
